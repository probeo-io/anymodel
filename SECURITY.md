# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in @probeo/anymodel, please report it responsibly.

**Do not open a public issue.**

Instead, email **security@probeo.io** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work with you on a fix before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Scope

This package routes API requests to third-party LLM providers. Security concerns may include:

- API key exposure or leakage
- Request/response data handling
- Config file parsing vulnerabilities
- Server mode endpoint security
