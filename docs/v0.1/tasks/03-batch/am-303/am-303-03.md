# AM-303-03: Return 400 if batch not completed

Story: [AM-303](../../stories/03-batch/am-303.md)
Status: todo

---

## Description

Guard against premature retrieval.

## Acceptance Criteria

- [ ] Returns 400 with message "Batch is still processing" if status is pending or processing
- [ ] Returns results normally for completed, failed, or cancelled batches
- [ ] Failed/cancelled batches return partial results (whatever completed)
