# AM-312-01: Implement retention cleanup

Story: [AM-312](../../stories/03-batch/am-312.md)
Status: todo

---

## Description

Auto-delete old batches.

## Acceptance Criteria

- [ ] On startup, scan batch directory for completed/failed/cancelled batches
- [ ] Delete batches where created_at is older than retentionDays
- [ ] retentionDays configurable (default: 30)
- [ ] Active batches (pending/processing) never deleted
- [ ] Cleanup runs asynchronously (does not block startup)
