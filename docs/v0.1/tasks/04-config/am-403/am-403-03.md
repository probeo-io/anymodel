# AM-403-03: Implement ${ENV_VAR} interpolation

Story: [AM-403](../../stories/04-config/am-403.md)
Status: todo

---

## Description

Resolve env vars in config strings.

## Acceptance Criteria

- [ ] Scans all string values in config for ${VAR_NAME} pattern
- [ ] Replaces with process.env[VAR_NAME] value
- [ ] Undefined env var results in empty string (no error)
- [ ] Only applies to string values (not keys or non-string values)
- [ ] Recursive through nested objects and arrays
