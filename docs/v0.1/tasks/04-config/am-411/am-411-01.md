# AM-411-01: Apply batch config defaults

Story: [AM-411](../../stories/04-config/am-411.md)
Status: todo

---

## Description

Set default batch settings.

## Acceptance Criteria

- [ ] config.batch.pollInterval used as default poll interval (default: 30000)
- [ ] config.batch.concurrencyFallback used as concurrent limit (default: 10)
- [ ] config.batch.dir used as batch storage directory
- [ ] config.batch.retentionDays used for auto-cleanup (default: 30)
- [ ] All overridable per-batch via SDK options
