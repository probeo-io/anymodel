# AM-211-02: Skip retry when fallback models available

Story: [AM-211](../../stories/02-providers/am-211.md)
Status: todo

---

## Description

Prefer fallback over retry.

## Acceptance Criteria

- [ ] When models[] fallback array is set, do not retry failed model
- [ ] Instead, immediately advance to next model in the array
- [ ] Retry logic only applies when no fallback is configured
