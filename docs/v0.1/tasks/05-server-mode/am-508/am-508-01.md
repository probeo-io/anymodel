Story: [AM-508](../../stories/05-server-mode/am-508.md)
Status: todo

---

## Description

Bearer token authentication.

## Acceptance Criteria

- [ ] When ANYMODEL_SERVER_KEY is set, middleware runs on all routes except /health
- [ ] Checks Authorization header for "Bearer {key}" format
- [ ] Returns 401 { error: { code: 401, message: "Invalid API key" } } if key doesn't match
- [ ] Returns 401 if Authorization header missing
- [ ] Passes through to handler if key matches
