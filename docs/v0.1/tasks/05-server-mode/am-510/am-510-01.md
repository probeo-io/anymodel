## Description

Structured request logging.

## Acceptance Criteria

- [ ] Each request logged as one JSON line to stdout
- [ ] Fields: timestamp (ISO 8601), model, provider, prompt_tokens, completion_tokens, estimated_cost, latency_ms, status (success/error)
- [ ] Logged after response is sent (does not delay response)
- [ ] Log level "summary" is the default
