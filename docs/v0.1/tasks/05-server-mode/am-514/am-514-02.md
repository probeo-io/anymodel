# AM-514-02: Graceful shutdown on SIGTERM

Story: [AM-514](../../stories/05-server-mode/am-514.md)
Status: todo

---

## Description

Clean shutdown for containers.

## Acceptance Criteria

- [ ] Listens for SIGTERM and SIGINT signals
- [ ] Stops accepting new connections
- [ ] Waits for in-flight requests to complete (up to 30s timeout)
- [ ] Logs "Server shutting down..." message
- [ ] Exits with code 0 on clean shutdown
