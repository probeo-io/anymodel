# AM-111-03: Calculate cost from pricing metadata

Story: [AM-111](../../stories/01-core-api/am-111.md)
Status: todo

---

## Description

Compute total_cost per generation.

## Acceptance Criteria

- [ ] Reads prompt and completion pricing from model metadata
- [ ] total_cost = (prompt_tokens * prompt_price) + (completion_tokens * completion_price)
- [ ] Returns 0 cost if pricing metadata unavailable (custom providers)
- [ ] Cost in USD as a number (e.g., 0.0042)
