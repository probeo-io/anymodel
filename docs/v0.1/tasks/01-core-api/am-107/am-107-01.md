# AM-107-01: Implement middle-out transform

Story: [AM-107](../../stories/01-core-api/am-107.md)
Status: todo

---

## Description

Truncate middle messages to fit context window.

## Acceptance Criteria

- [ ] Calculates total token count of messages
- [ ] When tokens exceed model context_length, removes messages from the middle
- [ ] System prompt (first message if role: system) is always preserved
- [ ] Last N messages are always preserved (N configurable, default 4)
- [ ] Removed messages replaced with a single system note indicating truncation
- [ ] Transform runs before request is sent to adapter
