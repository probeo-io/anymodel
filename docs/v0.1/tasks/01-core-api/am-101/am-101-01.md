# AM-101-01: Define ChatCompletion TypeScript types

Story: [AM-101](../../stories/01-core-api/am-101.md)
Status: todo

---

## Description

Create TypeScript interfaces for ChatCompletionRequest, ChatCompletion, ChatCompletionChoice, Message, Usage matching the OpenAI spec.

## Acceptance Criteria

- [ ] ChatCompletionRequest interface has required `model` (string) and `messages` (Message[]) fields
- [ ] ChatCompletionRequest has optional fields: temperature, max_tokens, top_p, top_k, frequency_penalty, presence_penalty, repetition_penalty, seed, stop, stream, logprobs, top_logprobs, response_format, tools, tool_choice, user
- [ ] ChatCompletion interface has id (string), object ("chat.completion"), created (number), model (string), choices (ChatCompletionChoice[]), usage (Usage)
- [ ] Usage interface has prompt_tokens, completion_tokens, total_tokens (all number)
- [ ] ChatCompletionChoice has index, message (Message), finish_reason (string)
- [ ] Message interface has role ("system"|"user"|"assistant"|"tool") and content (string), optional tool_calls
- [ ] All types exported from package index
