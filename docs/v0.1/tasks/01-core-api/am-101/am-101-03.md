## Description

Create a utility that generates unique gen- prefixed IDs for every completion response.

## Acceptance Criteria

- [ ] IDs follow format gen-{random alphanumeric string}
- [ ] IDs are unique across concurrent requests
- [ ] Minimum 16 characters of randomness
- [ ] Function is synchronous and has no external dependencies
