Story: [AM-101](../../stories/01-core-api/am-101.md)
Status: todo

---

## Description

Build the main SDK method that accepts a ChatCompletionRequest, resolves the provider from the model string, delegates to the adapter, and returns a ChatCompletion.

## Acceptance Criteria

- [ ] Accepts ChatCompletionRequest, returns Promise<ChatCompletion> (or async iterable when stream: true)
- [ ] Parses model string to extract provider slug and model ID
- [ ] Resolves alias if model matches an alias before provider lookup
- [ ] Retrieves the correct ProviderAdapter from the registry
- [ ] Calls adapter.translateRequest(), sends to provider, calls adapter.translateResponse()
- [ ] Returns normalized ChatCompletion with gen- prefixed ID
