## AM-309: Webhook notification
**As a** developer
**I want to** specify a `webhook` URL on batch creation
**So that** I get notified when a batch completes without polling

**Acceptance Criteria:**
- `webhook` field on batch create request
- POST to webhook URL when batch reaches terminal status (completed/failed)
- Payload includes `id`, `status`, `total`, `completed`, `failed`, `results_url`
- Webhook fires from the server process (server mode only)
- Retries webhook delivery on failure (3 attempts with backoff)
