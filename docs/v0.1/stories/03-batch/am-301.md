## AM-301: Create a batch
**As a** developer
**I want to** submit a batch of chat completion requests via `POST /batches`
**So that** I can process many prompts asynchronously at 50% cost

**Acceptance Criteria:**
- Accepts `model`, `requests[]` (each with `custom_id` and `messages`), and optional shared `options`
- Per-request params override shared `options`
- Returns batch object with `id`, `status: "pending"`, `batch_mode`, `total`, timestamps
- `batch_mode` is `"native"` for Anthropic/OpenAI, `"concurrent"` for others
- Batch state persisted to `~/.anymodel/batches/{id}/` immediately
- `requests.jsonl` and `meta.json` written on create
- SDK: `client.batches.create({ model, requests, options })`
