## AM-306: Manual polling loop
**As a** developer
**I want to** poll `client.batches.retrieve(id)` in a loop
**So that** I have full control over polling interval and logic

**Acceptance Criteria:**
- `retrieve()` returns fresh status each call
- For native providers, queries upstream API to sync status
- `completed` and `failed` counters reflect current progress
- `status` transitions: `pending → processing → completed/failed/cancelled`
