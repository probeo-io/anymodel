## AM-304: Cancel a batch
**As a** developer
**I want to** cancel a running batch via `DELETE /batches/:id`
**So that** I can stop processing and avoid unnecessary cost

**Acceptance Criteria:**
- Sends cancel to upstream provider (Anthropic: cancel batch, OpenAI: cancel batch)
- Status transitions to `"cancelled"`
- Already-completed results are preserved and retrievable
- For concurrent fallback, stops dispatching new requests
- SDK: `client.batches.cancel(id)`
