## AM-303: Retrieve batch results
**As a** developer
**I want to** get completed results via `GET /batches/:id/results`
**So that** I can process the model responses

**Acceptance Criteria:**
- Returns `results[]` with `custom_id`, `status` (success/error), `response` (ChatCompletion), `error`
- Includes `usage_summary` with total tokens and estimated cost
- Each `response` is the same shape as a `/chat/completions` response
- Results read from `results.jsonl` on disk
- Returns 400 if batch is not yet completed
- SDK: `client.batches.results(id)`
