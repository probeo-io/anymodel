## AM-311: Output directory for results
**As a** developer
**I want** to specify an `outputDir` that writes one JSON file per `custom_id`
**So that** I can consume results as individual files in my pipeline

**Acceptance Criteria:**
- `outputDir` option on `createAndPoll()`
- Creates `{outputDir}/{custom_id}.json` for each completed result
- Each file is a full `ChatCompletion` response object
- `{outputDir}/_batch_summary.json` written with batch metadata and usage summary
- Directory created if it doesn't exist
