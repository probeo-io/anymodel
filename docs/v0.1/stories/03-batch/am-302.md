## AM-302: Retrieve batch status
**As a** developer
**I want to** check batch progress via `GET /batches/:id`
**So that** I can monitor completion and show progress to users

**Acceptance Criteria:**
- Returns batch object with current `status`, `completed`, `failed`, `total` counts
- Counters update as individual requests finish
- Reads from `meta.json` on disk (works across process restarts)
- For native providers, syncs with upstream status on each retrieve call
- SDK: `client.batches.retrieve(id)`
