## AM-307: createAndPoll helper
**As a** developer
**I want** a `client.batches.createAndPoll()` that creates and waits for completion
**So that** I don't have to write my own polling loop

**Acceptance Criteria:**
- Creates batch and polls automatically until terminal status
- Configurable `pollInterval` (default: 30s)
- `timeout: 0` means wait indefinitely (default)
- `onProgress` callback fires on each poll with current status
- Returns the full results when complete
- Throws on batch-level failure
