## AM-314: OpenAI batch adapter
**As a** developer using `openai/*` models
**I want** batches to use OpenAI's native Batch API
**So that** I get the 50% batch discount

**Acceptance Criteria:**
- Translates requests to JSONL format
- Uploads JSONL via Files API
- Creates batch via `POST /v1/batches` with file ID
- Polls via `GET /v1/batches/:id`
- Downloads output file when complete, parses JSONL results
- Maps results back to unified format (already OpenAI shape, just needs `gen-` prefix)
- `batch_mode: "native"` in response
