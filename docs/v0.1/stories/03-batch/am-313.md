## AM-313: Anthropic batch adapter
**As a** developer using `anthropic/*` models
**I want** batches to use Anthropic's native Message Batches API
**So that** I get the 50% batch discount

**Acceptance Criteria:**
- Translates requests to Anthropic batch format (`custom_id` + `params`)
- Submits via `POST /v1/messages/batches`
- Polls via `GET /v1/messages/batches/:id` (checks `processing_status`)
- Downloads results from `results_url` when `processing_status: "ended"`
- Maps each result back to unified `ChatCompletion` format
- `batch_mode: "native"` in response
- Handles Anthropic-specific statuses: `succeeded`, `errored`, `canceled`, `expired`
