## AM-305: List batches
**As a** developer
**I want to** list all batches via `GET /batches`
**So that** I can see all past and in-progress batches

**Acceptance Criteria:**
- Returns all batches from disk (sorted by `created_at` descending)
- Includes status, progress counts, model, batch_mode
- Supports pagination (limit/offset)
- SDK: `client.batches.list()`
