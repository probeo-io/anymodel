## AM-205: OpenAI pass-through adapter
**As a** developer using `openai/*` models
**I want** requests forwarded to OpenAI with minimal transformation
**So that** all OpenAI features work natively

**Acceptance Criteria:**
- Request body forwarded as-is (already in correct format)
- Response `id` re-prefixed with `gen-` for consistency
- Errors mapped to standardized error format
- All OpenAI-specific parameters preserved (logit_bias, logprobs, etc.)
