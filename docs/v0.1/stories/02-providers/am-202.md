## AM-202: Anthropic request translation
**As a** developer using `anthropic/claude-*` models
**I want** my OpenAI-format request to be translated to Anthropic's Messages API
**So that** I don't need to learn Anthropic's native format

**Acceptance Criteria:**
- System message extracted from `messages` array to top-level `system` parameter
- `max_tokens` defaults to 4096 when not specified (required by Anthropic)
- `tools` translated to Anthropic tool format
- `response_format: { type: "json_object" }` handled appropriately
- `top_k` passed through natively
- Unsupported parameters stripped silently
