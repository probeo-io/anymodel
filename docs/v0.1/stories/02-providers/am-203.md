## AM-203: Anthropic response normalization
**As a** developer
**I want** Anthropic responses normalized to OpenAI `ChatCompletion` format
**So that** I can process responses identically regardless of provider

**Acceptance Criteria:**
- Anthropic `content[].text` mapped to `choices[].message.content`
- Anthropic `stop_reason` mapped to `finish_reason` (`end_turn` → `stop`, `max_tokens` → `length`, `tool_use` → `tool_calls`)
- `usage.input_tokens` → `prompt_tokens`, `usage.output_tokens` → `completion_tokens`
- `id` prefixed with `gen-`
- Tool use responses mapped to `tool_calls` array
