## AM-206: Gemini request translation
**As a** developer using `google/gemini-*` models
**I want** my OpenAI-format request translated to Gemini's API
**So that** I can use Gemini without learning its native format

**Acceptance Criteria:**
- `messages` mapped to Gemini `contents` format (role mapping: `assistant` → `model`)
- System message mapped to Gemini's system instruction
- `tools` mapped to Gemini function declarations
- `response_format` mapped to Gemini's response MIME type / schema
- `max_tokens` mapped to `maxOutputTokens`
