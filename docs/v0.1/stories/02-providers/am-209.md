## AM-209: Custom provider registration
**As a** developer
**I want** to add any OpenAI-compatible endpoint (Groq, Together, Ollama, vLLM)
**So that** I can use any model from any OpenAI-compatible service

**Acceptance Criteria:**
- Configured via `providers.custom` in config with `baseURL` and optional `apiKey`
- Referenced as `custom-name/model-name` in requests
- Auto-discovers models via `GET /models` on the custom endpoint
- Pass-through adapter (same as OpenAI but with custom baseURL)
- Works for local models (Ollama, vLLM) with no auth
