## AM-204: Anthropic streaming translation
**As a** developer
**I want** Anthropic SSE events translated to OpenAI delta chunks
**So that** streaming works identically across providers

**Acceptance Criteria:**
- `content_block_delta` events → `delta.content` chunks
- `message_delta` with `stop_reason` → `finish_reason` in final chunk
- `message_start` with `usage` → usage in final chunk
- Tool use streaming translated to `delta.tool_calls`
