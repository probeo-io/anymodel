## AM-214: Provider preference routing
**As a** developer
**I want** to set `provider.order`, `provider.only`, `provider.ignore`
**So that** I can control which providers handle my requests

**Acceptance Criteria:**
- `order` — tries providers in this sequence
- `only` — restricts to these providers only
- `ignore` — excludes these providers
- `allow_fallbacks: false` — fails if preferred provider unavailable
- `require_parameters: true` — only routes to providers supporting all requested params
