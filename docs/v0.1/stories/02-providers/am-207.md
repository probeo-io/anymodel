## AM-207: Gemini response normalization
**As a** developer
**I want** Gemini responses normalized to OpenAI format

**Acceptance Criteria:**
- Gemini `candidates[].content.parts[].text` → `choices[].message.content`
- Gemini finish reasons mapped to OpenAI equivalents
- Function call responses mapped to `tool_calls`
- Usage stats mapped (Gemini uses different field names)
