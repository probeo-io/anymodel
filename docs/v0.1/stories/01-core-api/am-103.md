## AM-103: Tool/function calling
**As a** developer
**I want to** pass `tools` and `tool_choice` in my request
**So that** the model can invoke functions in my application

**Acceptance Criteria:**
- `tools` array accepted with OpenAI-compatible function definitions
- `tool_choice` accepts `"none"`, `"auto"`, `"required"`, or `{ type: "function", function: { name } }`
- Response `choices[].message.tool_calls` populated when model invokes a tool
- `finish_reason: "tool_calls"` when the model wants to call a function
- Works across all providers (Anthropic tools translated, OpenAI pass-through, Gemini mapped)
