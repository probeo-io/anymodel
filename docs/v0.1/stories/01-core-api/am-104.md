## AM-104: Structured output / response format
**As a** developer
**I want to** request JSON output via `response_format`
**So that** I get structured, parseable responses

**Acceptance Criteria:**
- `response_format: { type: "json_object" }` forces JSON output
- `response_format: { type: "json_schema", json_schema: {...} }` validates against a schema
- `response_format: { type: "text" }` is the default
- Translated appropriately per provider (Anthropic, OpenAI, Gemini each handle differently)
