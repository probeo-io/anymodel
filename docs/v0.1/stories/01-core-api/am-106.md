## AM-106: Provider preferences
**As a** developer
**I want to** set `provider` preferences on a request
**So that** I can control which providers are used and in what order

**Acceptance Criteria:**
- `provider.order` — tries providers in specified order
- `provider.only` — restricts to allowlisted providers
- `provider.ignore` — excludes blocklisted providers
- `provider.allow_fallbacks` — when false, fails instead of trying alternates
- `provider.require_parameters` — only routes to providers supporting all params in request
- `provider.sort` — sorts by `"price"`, `"throughput"`, or `"latency"`
