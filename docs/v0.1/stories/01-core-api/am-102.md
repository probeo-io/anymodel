## AM-102: Streaming completions
**As a** developer
**I want to** stream a chat completion response via SSE
**So that** I can display tokens to the user as they arrive

**Acceptance Criteria:**
- `stream: true` returns an async iterable of `ChatCompletionChunk` objects
- Each chunk has `delta.content` with the token text
- Final chunk includes `finish_reason` and `usage`
- Stream ends with `data: [DONE]`
- Keepalive comments (`: ANYMODEL PROCESSING`) sent during provider processing delays
- Mid-stream errors return HTTP 200 with `finish_reason: "error"` in the chunk
