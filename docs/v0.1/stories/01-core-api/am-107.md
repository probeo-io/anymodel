## AM-107: Middle-out transform
**As a** developer
**I want to** use `transforms: ["middle-out"]`
**So that** long conversations are truncated to fit the model's context window without losing the system prompt or most recent messages

**Acceptance Criteria:**
- When total tokens exceed model context length, middle messages are dropped
- System prompt is always preserved
- Most recent N messages are always preserved
- Truncation happens before the request is sent to the provider
- Works transparently — caller doesn't need to know the context was truncated
