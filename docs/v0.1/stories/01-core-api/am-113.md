## AM-113: TypeScript SDK client
**As a** developer
**I want** an `AnyModel` class with the same interface as the OpenAI SDK
**So that** I can drop it into existing codebases with minimal changes

**Acceptance Criteria:**
- `new AnyModel({ anthropic: { apiKey }, openai: { apiKey } })` instantiation
- `client.chat.completions.create()` — same signature as OpenAI SDK
- `client.models.list()` — model discovery
- `client.generation.get()` — generation stats
- Streaming via async iterators
- Full TypeScript types for all request/response objects
- Published as `@probeo/anymodel` on npm
