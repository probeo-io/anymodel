## AM-101: Basic chat completion
**As a** developer using the SDK
**I want to** send a chat completion request with `model` and `messages`
**So that** I get an OpenAI-compatible response back regardless of the underlying provider

**Acceptance Criteria:**
- `client.chat.completions.create({ model: 'anthropic/claude-sonnet-4-6', messages: [...] })` returns a `ChatCompletion` object
- Response includes `id` (gen- prefix), `object`, `created`, `model`, `choices`, `usage`
- `usage` contains `prompt_tokens`, `completion_tokens`, `total_tokens`
- All standard parameters accepted: `temperature`, `max_tokens`, `top_p`, `top_k`, `frequency_penalty`, `presence_penalty`, `repetition_penalty`, `seed`, `stop`
