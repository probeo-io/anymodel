## AM-105: Model fallback routing
**As a** developer
**I want to** specify a `models` array with `route: "fallback"`
**So that** if my primary model fails, the request automatically tries the next model

**Acceptance Criteria:**
- `models: ['anthropic/claude-sonnet-4-6', 'openai/gpt-4o']` with `route: 'fallback'` tries models in order
- On provider error/timeout, moves to next model without the caller retrying
- Response `model` field reflects which model actually served the request
- If all models fail, returns the last error
