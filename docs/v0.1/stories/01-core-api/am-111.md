## AM-111: Generation stats endpoint
**As a** developer
**I want to** query `GET /generation?id=gen-xxx`
**So that** I can get cost, usage, and latency details for a completed request

**Acceptance Criteria:**
- Returns: `id`, `model`, `provider_name`, `total_cost`, `tokens_prompt`, `tokens_completion`, `latency`, `generation_time`, `created_at`, `finish_reason`, `streamed`
- `total_cost` calculated from model pricing metadata
- Stats persisted across process restarts (stored on disk)
- SDK: `client.generation.get('gen-xxx')` returns the same data
