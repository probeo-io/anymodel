## AM-108: List available models
**As a** developer
**I want to** call `GET /models` or `client.models.list()`
**So that** I can discover what models are available across my configured providers

**Acceptance Criteria:**
- Returns `{ data: ModelInfo[] }` with all models from all configured providers
- Each model includes: `id`, `name`, `created`, `description`, `context_length`, `pricing`, `architecture`, `top_provider`, `supported_parameters`
- `pricing.prompt` and `pricing.completion` are strings (cost per token)
- Can filter by provider: `client.models.list({ provider: 'anthropic' })`
- Custom OpenAI-compatible providers included (auto-discovered via their `/models` endpoint)
