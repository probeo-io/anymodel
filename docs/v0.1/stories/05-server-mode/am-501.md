## AM-501: Start HTTP server via CLI
**As an** operator
**I want to** run `npx @probeo/anymodel serve`
**So that** I get an HTTP server that proxies requests to any configured provider

**Acceptance Criteria:**
- Starts HTTP server on port 4141 (default)
- Loads config (env vars → global config → local config)
- Logs startup info: port, configured providers, number of available models
- Exits with clear error if no providers are configured
