## AM-511: Full logging
**As a** developer debugging
**I want** full request/response bodies logged
**So that** I can see exactly what was sent to and received from the provider

**Acceptance Criteria:**
- `--log full` or `ANYMODEL_LOG_LEVEL=full`
- Logs full request body (messages, tools, params)
- Logs full response body (choices, usage)
- Streaming requests log the assembled full response
- Warning: may contain sensitive data
