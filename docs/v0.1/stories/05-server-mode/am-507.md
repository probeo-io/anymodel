## AM-507: Model endpoints info
**As a** client application
**I want** `GET /api/v1/models/:provider/:model/endpoints`
**So that** I can see which provider endpoints serve a specific model

**Acceptance Criteria:**
- Returns endpoint details for the specified model
- Includes pricing, context length, max completion tokens per endpoint
- Returns 404 for unknown models
