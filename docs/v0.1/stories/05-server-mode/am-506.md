## AM-506: Generation stats endpoint
**As a** client application
**I want** `GET /api/v1/generation?id=gen-xxx`
**So that** I can get cost and usage details for a completed request

**Acceptance Criteria:**
- Returns generation stats by ID
- Stats persisted on disk (survive server restarts)
- Returns 404 for unknown generation IDs
