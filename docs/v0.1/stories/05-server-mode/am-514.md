## AM-514: Docker sidecar deployment
**As a** DevOps engineer
**I want** to run anymodel as a Docker sidecar
**So that** my application containers can hit it over the local network

**Acceptance Criteria:**
- Server binds to `0.0.0.0` by default (accessible from other containers)
- Configurable via env vars only (no config file needed)
- Graceful shutdown on SIGTERM
- Health check endpoint: `GET /health` returns 200
