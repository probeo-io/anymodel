## AM-503: Chat completions endpoint
**As a** client application
**I want** `POST /api/v1/chat/completions` to work
**So that** I can use any OpenAI SDK pointed at this server

**Acceptance Criteria:**
- Full request/response format as defined in 01-core-api
- Streaming via SSE when `stream: true`
- Keepalive comments during processing
- Error responses with proper HTTP status codes
- CORS headers for browser-based clients
