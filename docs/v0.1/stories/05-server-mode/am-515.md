## AM-515: Health check
**As an** operator
**I want** a health check endpoint
**So that** load balancers and orchestrators can monitor server health

**Acceptance Criteria:**
- `GET /health` returns `200 OK` with `{ "status": "ok" }`
- Does not require authentication
- Does not require provider connectivity (checks server process only)
