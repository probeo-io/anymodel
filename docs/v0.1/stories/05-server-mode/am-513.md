## AM-513: Drop-in OpenAI SDK usage
**As a** developer
**I want** to point the OpenAI SDK at the anymodel server with just `baseURL`
**So that** I can use anymodel without changing any client code

**Acceptance Criteria:**
- `new OpenAI({ baseURL: 'http://localhost:4141/api/v1', apiKey: '...' })` works
- `client.chat.completions.create()` works (non-streaming and streaming)
- `client.models.list()` works
- `provider/model` naming works as the model parameter
- All standard OpenAI parameters supported
