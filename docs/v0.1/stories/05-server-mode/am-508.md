## AM-508: Server API key auth
**As an** operator
**I want** to set `ANYMODEL_SERVER_KEY` to require authentication
**So that** the server isn't open to unauthorized access

**Acceptance Criteria:**
- When `ANYMODEL_SERVER_KEY` is set, all requests require `Authorization: Bearer <key>`
- Returns 401 for missing or invalid key
- When not set, all requests are accepted (for local/Docker use)
- Auth check happens before any processing
