## AM-509: Open mode (no auth)
**As a** developer running locally
**I want** the server to work without setting up auth
**So that** I can get started with zero config beyond provider keys

**Acceptance Criteria:**
- No `ANYMODEL_SERVER_KEY` → all requests accepted
- Warning logged on startup: "Server running without authentication"
