## AM-502: CLI options
**As an** operator
**I want** CLI flags for port, host, config path, and log level
**So that** I can customize the server without editing config files

**Acceptance Criteria:**
- `--port <number>` — override port (default: 4141)
- `--host <string>` — bind address (default: 0.0.0.0)
- `--config <path>` — path to config file
- `--log <level>` — off, summary, full (default: summary)
- CLI flags override env vars and config file
