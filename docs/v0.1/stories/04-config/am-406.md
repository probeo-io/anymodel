## AM-406: Built-in provider config
**As a** developer
**I want** to configure Anthropic, OpenAI, and Google with API keys and default models
**So that** I can customize per-provider behavior

**Acceptance Criteria:**
- Each provider accepts: `apiKey`, `defaultModel`
- `defaultModel` used when no model specified in `provider/` prefix
- Provider only registered if `apiKey` is present (config or env var)
