## AM-413: Model discovery via SDK
**As a** developer
**I want** to list all available models programmatically
**So that** I can build model selectors and validate model references

**Acceptance Criteria:**
- `client.models.list()` returns all models from all configured providers
- `client.models.list({ provider: 'anthropic' })` filters by provider
- Response matches OpenRouter model format (id, name, pricing, context_length, etc.)
- Custom providers included (auto-discovered or manually listed)
- Cached after first call (refreshable via `client.models.refresh()`)
