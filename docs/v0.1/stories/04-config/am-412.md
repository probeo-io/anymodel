## AM-412: Transforms
**As a** developer
**I want** to configure default transforms
**So that** context window management happens automatically

**Acceptance Criteria:**
- `transforms: ["middle-out"]` supported in config defaults
- Also settable per-request
- `"middle-out"` truncates middle messages to fit context window
- System prompt and recent messages always preserved
