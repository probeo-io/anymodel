## AM-409: Default request parameters
**As a** developer
**I want** to set default values for `temperature`, `max_tokens`, etc.
**So that** I don't have to specify them on every request

**Acceptance Criteria:**
- Config `defaults` section: `temperature`, `max_tokens`, `retries`, `timeout`
- Applied to all requests unless overridden per-request
- Provider-required defaults (e.g., Anthropic's `max_tokens`) still applied if not in config
