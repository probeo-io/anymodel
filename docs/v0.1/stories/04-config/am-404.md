## AM-404: Global config file
**As a** developer
**I want** a global config at `~/.anymodel/config.json`
**So that** my defaults apply across all projects

**Acceptance Criteria:**
- Falls back to `~/.anymodel/config.json` if no local config found
- Local config overrides global config
- Programmatic options override both
