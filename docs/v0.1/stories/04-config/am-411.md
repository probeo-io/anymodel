## AM-411: Batch settings
**As a** developer
**I want** to configure batch behavior globally
**So that** I don't have to specify polling interval and concurrency on every batch

**Acceptance Criteria:**
- `batch.pollInterval` — ms between polls (default: 30000)
- `batch.concurrencyFallback` — concurrent requests for non-native batch (default: 10)
- `batch.dir` — storage directory (default: `~/.anymodel/batches/`)
- `batch.retentionDays` — auto-cleanup threshold (default: 30)
- All overridable per-batch via SDK options
