## AM-402: Server env vars
**As an** operator
**I want** to configure server settings via environment variables
**So that** I can configure the server in Docker/CI without config files

**Acceptance Criteria:**
- `ANYMODEL_PORT` — server port (default: 4141)
- `ANYMODEL_SERVER_KEY` — optional auth key for server mode
- `ANYMODEL_LOG_LEVEL` — `off`, `summary`, `full`
- `ANYMODEL_BATCH_DIR` — batch storage directory
