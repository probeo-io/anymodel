## AM-410: Default routing config
**As an** operator
**I want** to configure default routing behavior (fallback order, allow_fallbacks)
**So that** all requests follow my preferred routing strategy

**Acceptance Criteria:**
- `routing.fallback_order` — default provider order for fallbacks
- `routing.allow_fallbacks` — global default (can be overridden per-request)
- Per-request `provider` preferences override config-level routing
